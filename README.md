# Gamer.com E-commerce Backend

## Run Locally

### 1. Clone repo

```
$ git clone git@github.com:scarcrux/tpp-capstone-backend.git
$ cd tpp-capstone-backend
```

### 2. Install MongoDB

Download it from here: https://docs.mongodb.com/manual/administration/install-community/

### 3. Run Backend

```
$ npm install
$ npm start
